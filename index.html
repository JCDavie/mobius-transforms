The Möbius transformation is f( z ) = ( az + b ) / ( cz + d ) where z, a, b, c and d are complex numbers. Drag the points to change the transformation.

(Canvas drawing not supported by your browser.)

To simplify the transform, set c to zero (move it to the origin). Now a and b between them give a similarity transform: a controls the scale and rotation while b controls the translation.

The identity transform is given when b=c=0 and a=d.

A normalized Möbius transformation is one where ad - bc = 1.
